## 운영 방식

```text
notes/inbox/
→ 아직 정리 전인 원문, 아이디어, 질문을 임시 보관한다.

courses/<topic>/legacy/
→ 블로그 원문이나 예전 자료를 보존한다.

courses/<topic>/docs/
→ 다시 읽기 좋은 교재형 문서만 둔다.
```

원문은 바로 고치지 않는다.

먼저 `legacy/`나 `notes/inbox/`에 보관하고, 교재로 다듬은 글만 `docs/`에 둔다.
