# React

React 학습 자료를 교재형 문서로 정리할 공간.

아직 블로그 원문은 가져오지 않았다.

## 다음 작업

사용자가 React 블로그 학습 자료 위치를 알려주면 시작한다.

순서:

```text
1. 원문 파일을 legacy/blog-import/에 보관
2. 파일 목록과 제목 파악
3. 주제별 묶음으로 분류
4. docs/에 교재형 문서로 리라이트
5. README 목차 갱신
```

대량 자료는 한 번에 다 읽지 않는다.

먼저 목록과 분량을 보고 묶음 단위로 처리한다.

## 구조

```text
docs/
→ 다시 읽기 좋은 교재형 문서

labs/
→ 실습 코드, 예제, 관찰 기록

legacy/blog-import/
→ 블로그에서 가져온 원문 보관
```

## 예정

블로그 학습 글을 먼저 `legacy/blog-import/`에 보관한다.

그 다음 주제별로 묶어서 `docs/`에 리라이트한다.
